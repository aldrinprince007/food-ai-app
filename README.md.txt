# 🍔 AI Food Deal Finder

An AI-powered web app that helps users find the best food options and deals across Swiggy, Zomato, and Blinkit.

## 🚀 Features

* 🔍 Search food by name
* 💰 Filter by budget
* ⭐ Compare ratings
* 🎁 View offers
* 🧠 AI-based best recommendation

## 🛠️ Tech Stack

* Python
* Streamlit
* OpenAI API

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 🔐 Environment Variables

Create a `.env` file and add:
OPENAI_API_KEY=your_api_key_here
